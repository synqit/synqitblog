## Step-by-step: creating a new user

### Step 1  
Go to **https://admin.microsoft.com** and sign in with your administrator account.

### Step 2  
In the left-hand menu, select **Users**, and then choose **Active users**.

### Step 3  
Just below the *Active users* heading, select **Add user**.

### Step 4  
Fill out the basic information:
- First name  
- Last name  
- Display name  
- Username  

Make sure you select the correct domain for the user from the dropdown list.

### Step 5  
Choose how the password should be created:
- Automatically generate a password, or  
- Set a password manually  

You can also choose whether the user must change their password at first sign-in.

### Step 6  
Select the correct **location** and assign a **product license**.  
You *can* continue without assigning a license, but this is usually not recommended.

### Step 7  
Under **Optional settings**, decide whether:
- The user should be a standard user, or  
- The user should receive an admin role

### Step 8  
Review all settings on the final page and click **Finish adding**.
